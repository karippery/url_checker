# URL Checker

A React-based web application that validates URL formats and checks their existence in real-time. Built with TypeScript, Material-UI, and modern React patterns.

## Features

- **Real-time URL Validation** - Instant format validation as you type
- **Existence Checking** - Mock server integration to check if URLs exist
- **Debounced Requests** - Optimized API calls with 500ms debouncing
- **Responsive Design** - Clean Material-UI interface
- **Type Safety** - Full TypeScript support

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **UI Framework**: Material-UI (MUI)
- **Testing**: Vitest + React Testing Library
- **Package Manager**: npm

## Installation


1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Open in browser**
   ```
   http://localhost:5173
   ```
